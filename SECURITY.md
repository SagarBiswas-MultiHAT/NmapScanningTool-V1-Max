# Security Policy

## Supported Versions

Security fixes are applied to the latest version on the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, do not open a public GitHub issue.

1. Email: `security@multihat.dev`
2. Include: affected version, reproduction steps, potential impact, and suggested remediation if available.
3. You will receive an acknowledgment within 3 business days.

## Disclosure Process

- We validate the report and assess severity.
- We coordinate a fix and release timeline.
- We publish remediation notes after the fix is available.

## Safe Usage Note

This tool is for authorized security testing only. Unauthorized scanning may violate law or policy.
